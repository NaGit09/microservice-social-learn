# Shared UI/UX Pro Max Assets

This directory serves as a repository for premium design assets, components, and tokens shared across the application to maintain the "Pro Max" aesthetic.

## Contents

- **Icons**: Custom or premium icon sets.
- **Images**: High-quality background images, gradients, or textures.
- **Styles**: Shared CSS snippets or Tailwind configuration extensions.
- **Components**: Reusable Vue components that don't fit into the main component library yet.

## Usage

Reference files in this directory using valid relative paths or import aliases if configured.
